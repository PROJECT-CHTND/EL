# EL Demo Frontend

ELのデモ用フロントエンドです。EL APIと連携して、インタビューセッションの実行やナレッジマップの可視化を行います。

## 技術スタック

- React 18+
- TypeScript
- Tailwind CSS

## セットアップ

```bash
npm install
npm run dev
```

`http://localhost:3000` でアクセスできます。

## 環境変数

| 変数 | デフォルト | 説明 |
|------|-----------|------|
| `EL_API_URL` | `http://localhost:8000` | EL APIのベースURL |
